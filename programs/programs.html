Coding Problems & Algorithms

📌 Problem 1: Count Substrings Between X and Y

Problem Statement: Given a string, two characters X and Y, count the number of substrings that start with X and end with Y.

Java Solution:

public class CountSubstrings { public static int countSubstrings(String str, char x, char y) { int count = 0; int startIndex = -1; for (int i = 0; i < str.length(); i++) { char ch = str.charAt(i); // If current character is X, update startIndex if (ch == x) { startIndex = i; } // If current character is Y and startIndex is set, increment count else if (ch == y && startIndex != -1) { count += (i - startIndex); } } return count; } public static void main(String[] args) { String str = "abbcaceghcak"; char x = 'a'; char y = 'c'; System.out.println(countSubstrings(str, x, y)); // Output: 5 } }

📌 Problem 2: Character Count in String

Challenge: Count the number of characters in the string "today is thursday, tomorrow is friday" (excluding spaces).

Java Solution:

public class CharacterCount { public static void main(String[] args) { String str = "today is thursday, tomorrow is friday"; // Remove white spaces from the string str = str.replaceAll("\\s", ""); // Count the number of characters in the string int count = str.length(); // Print the count of characters System.out.println("Number of characters in the string: " + count); } }

Behavioral Interview Preparation

🏆 Amazon 16 Leadership Principles

The STAR method helps structure responses: Situation → Task → Action → Result

1. Customer Obsession

Focus on solving customer problems and improving user experience. Example: Conducting thorough testing during checkout flow improvements.

2. Bias for Action

Make quick decisions when problems arise. Example: Implementing emergency workarounds for critical defects before release.

3. Learn and Be Curious

Continuously expand knowledge. Example: Learning new automation frameworks and tools to improve efficiency.

4. Have Backbone; Disagree and Commit

Voice concerns professionally, then support decisions. Example: Suggesting alternative testing strategies with data-backed reasoning.

5. Strive to Be Earth's Best Employer

Foster positive team culture. Example: Organizing team workshops and improving collaboration.

6. Success and Scale Bring Broad Responsibility

Ensure scalability as projects grow. Example: Optimizing test automation framework for increased test cases.

⭐ STAR Response Example: Customer Obsession

SITUATION: Customers reported usability issues in the checkout process.
TASK: Thoroughly test the checkout flow and identify all issues.
ACTION: Conducted exploratory testing, collaborated with UX designers, simulated various scenarios.
RESULT: Identified and resolved issues before release, significantly improving user experience.

💪 Strengths & Weaknesses

Three Key Strengths:

  1. Attention to Detail - Meticulous approach helps identify subtle defects that could compromise quality. Consistent contributor to reliable deliverables.
  2. Adaptability - Easily adjust to changing requirements and new technologies. Quick learner who embraces new methodologies.
  3. Good Communication Skills - Clear articulation of complex testing issues to cross-functional teams. Effective collaboration with developers and stakeholders.

Areas for Improvement:

  1. Active Listening - Tendency to speak before fully hearing others. Solution: Practicing mindfulness and reflective listening techniques.
  2. Delegation - Prefer doing tasks independently rather than delegating. Solution: Actively working on trusting team members and distributing workload.
  3. Tactfulness - Sometimes lacking diplomacy in feedback. Solution: Being more mindful of communication delivery and perspective of others.

Sorting Algorithms

🔹 Bubble Sort

Time Complexity: O(n²) | Space Complexity: O(1) | Best For: Nearly sorted data
public class BubbleSort { public static void bubbleSort(int[] arr) { boolean swapped = true; int j = 0; int tmp; while (swapped) { swapped = false; j++; for (int i = 0; i < arr.length - j; i++) { if (arr[i] > arr[i + 1]) { tmp = arr[i]; arr[i] = arr[i + 1]; arr[i + 1] = tmp; swapped = true; } } } } public static void main(String[] args) { int arr[] = {3, 60, 35, 2, -45, 320, 5}; bubbleSort(arr); System.out.print("Sorted: "); for (int i : arr) { System.out.print(i + " "); } } }

🔹 Selection Sort

Time Complexity: O(n²) | Space Complexity: O(1) | Best For: When memory writes are costly
public class SelectionSort { public static int[] doSelectionSort(int[] arr) { for (int i = 0; i < arr.length - 1; i++) { int index = i; for (int j = i + 1; j < arr.length; j++) { if (arr[j] < arr[index]) { index = j; } } int smallerNumber = arr[index]; arr[index] = arr[i]; arr[i] = smallerNumber; } return arr; } public static void main(String[] args) { int[] arr1 = {10, 34, 2, 56, 7, 67, 88, 42}; int[] arr2 = doSelectionSort(arr1); for (int i : arr2) { System.out.print(i + " "); } } }

🔹 Insertion Sort

Time Complexity: O(n²) | Space Complexity: O(1) | Best For: Small or nearly sorted arrays
public class InsertionSort { public static int[] doInsertionSort(int[] input) { int temp; for (int i = 1; i < input.length; i++) { for (int j = i; j > 0; j--) { if (input[j] < input[j - 1]) { temp = input[j]; input[j] = input[j - 1]; input[j - 1] = temp; } } } return input; } public static void main(String[] args) { int[] arr1 = {10, 34, 2, 56, 7, 67, 88, 42}; int[] arr2 = doInsertionSort(arr1); for (int i : arr2) { System.out.print(i + " "); } } }
